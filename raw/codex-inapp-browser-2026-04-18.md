## 核心信息

- **功能**：Codex 新增内嵌浏览器，支持 Comment Mode
- **交互方式**：点击页面元素 → 自动截图 + 捕获 DOM 元素 → 作为精准上下文注入下一轮对话
- **解决的痛点**：
  - 不再需要在浏览器和编辑器之间切换
  - 不再需要手动拖截图
  - 不再需要描述不清楚的 Prompt
- **意义**：让 Agent 能直接"看到"UI，精确指向问题位置
