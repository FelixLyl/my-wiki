## 核心定位

**"An open-source & self-hostable Heroku / Netlify / Vercel alternative."**

Coolify 让你用自己的服务器（VPS、裸金属、树莓派等）获得云平台体验——管理服务器、应用程序和数据库，只需要 SSH 连接。

核心价值主张：
- **无供应商锁定** — 所有配置存储在自己的服务器上，停用 Coolify 后资源仍然运行
- **自主掌控** — 数据不离开自己的基础设施
- **云体验** — 有 Heroku/Vercel 类似的自动化和部署魔法

---

## 主要能力

- 管理 VPS、Bare Metal、Raspberry Pi 等任意服务器（通过 SSH）
- 一键部署应用、数据库、服务
- 支持 Docker Compose、Dockerfile、Nixpacks 等多种部署方式
- 自动 SSL 证书（Let's Encrypt）
- 实时日志、资源监控
- Git 推送触发自动部署（支持 GitHub/GitLab/Gitea）
- 多服务器/多团队管理

## 与 AI Agent 工作流的关联

Coolify 是 AI Agent 基础设施部署的重要工具，可用于：
- 自托管 OpenClaw、Claude Code 等 AI Agent 服务
- 部署 n8n（工作流自动化）、数据库等 Agent 依赖服务
- 替代 AWS/GCP/Vercel 等云平台，降低 AI 服务运行成本
- 支持私有化部署（数据不出境）

## 对主人工作的意义

作为 AI 工具落地负责人，Coolify 提供了一条**低成本私有化部署 AI 基础设施**的路径：
- 把 AI Agent、数据库、API 服务部署在自己的服务器上
- 控制成本（VPS 比 AWS 便宜）
- 保障数据安全（不经过第三方云）
- 简单运维（GUI + 自动化）
