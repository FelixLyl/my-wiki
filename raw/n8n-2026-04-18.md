## 核心定位

**"Secure Workflow Automation for Technical Teams"**

n8n 给技术团队提供「代码的灵活性 + no-code 的速度」——可以写 JavaScript/Python，也可以用可视化界面，400+ 集成，原生 AI 能力，完全控制数据和部署。

---

## 核心能力

- **代码 + No-Code 混合** — 写 JS/Python，加 npm 包，或用拖拽可视化界面
- **AI 原生** — 基于 LangChain 构建 AI Agent 工作流，支持自有数据和模型
- **完全控制** — 自托管（Fair-code license）或使用云版本
- **企业级** — 高级权限、SSO、离线部署
- **生态丰富** — 400+ 集成，900+ 现成工作流模板

## 快速启动

```bash
# npx 直接运行
npx n8n

# Docker
docker run -it --rm --name n8n -p 5678:5678 \
  -v n8n_data:/home/node/.n8n docker.n8n.io/n8nio/n8n
```

编辑器地址：http://localhost:5678

## 与 AI Agent 工作流的关联

n8n 是构建 AI 自动化管道的关键工具：
- 连接外部 API、数据库、SaaS 服务
- 作为 AI Agent 的工具调用层（HTTP 请求、数据处理、触发器）
- 内置 LangChain 节点，可搭建多步骤 AI 工作流
- 替代 Zapier/Make 在内部系统集成中的角色

---

## 对主人工作的意义

作为内部 AI 系统集成负责人：
- n8n 可以作为 AI Agent 与企业内部系统之间的**集成中间层**
- 替代手写 Webhook/集成代码，快速连接飞书、钉钉、数据库等
- 与 Coolify 配合，在自己的服务器上私有化部署 n8n
- 与调研课题 #003（Skill + Swagger 访问内部系统）有直接关联
