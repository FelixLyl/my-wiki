## 社区定位

Hermes Agent 被社区定位为 [[OpenClaw-Skill生态|OpenClaw]]（龙虾）上线以来第一个真正意义上的竞争对手。Prompt Engineer 宝玉（@dotey）安装试用后评价"还可以"，认为值得关注。

这一定位意味着：Hermes Agent 在功能覆盖或使用体验上已接近主流 Agent 框架的基准线，不再是边缘项目。
